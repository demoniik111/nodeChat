# nodeChat
Chat app with backend and frontend

## Used technologies
* Vanilla Javascript
* NodeJs
* Express for NodeJs
* SocketIO for permanent messaging

## How to use
```sh
$ npm install
$ cd /src/
$ node ./index.js
```
